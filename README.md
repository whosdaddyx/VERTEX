# VERTEX

VERTEX is an all-in-one Discord management platform built to help communities stay safer, organized, and easier to manage.

## Features

- Advanced moderation
- Intelligent AutoMod
- Verification
- Tickets
- Reaction roles
- Giveaways
- Welcome messages
- Music
- Server insights
- Twitch & YouTube notifications
