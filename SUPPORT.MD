# Support

Need help with VERTEX? We're here to help.

---

## Documentation

Before reaching out, we recommend checking the available documentation and setup guides.

Many common questions are already covered there.

---

## Bug Reports

If you've found a bug, please include as much information as possible:

* What happened
* What you expected to happen
* Steps to reproduce
* Screenshots (if applicable)
* Operating system
* Browser (for dashboard issues)
* Discord client (Desktop, Web, or Mobile)

Providing detailed information helps us resolve issues much faster.

---

## Feature Requests

Have an idea that could improve VERTEX?

We welcome feature suggestions from the community.

When submitting a request, please explain:

* The problem you're trying to solve
* Your proposed solution
* Why it would benefit Discord communities

---

## Security Issues

If you've discovered a potential security vulnerability, please **do not** open a public issue.

Instead, report it privately following the instructions in **SECURITY.md**.

Responsible disclosure helps protect all users.
